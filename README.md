# YF 나의 전용 문의 (핸드폰 웹앱)

쏘나타 **YF 2010 · LPG(LPI) 2.0 · 단거리 왕복 15km 출퇴근차** 전용 엔진오일 교환 가이드를
핸드폰 홈 화면에 앱처럼 추가해 쓰는 PWA입니다. (데스크톱 "YF 중고차 부품도감 → 나의 전용 문의"의 모바일판)

## 쓰는 법 (핸드폰 홈에 앱으로 추가)
1. 폰 브라우저(크롬/사파리)로 배포 주소 열기 → `https://donglee1228.github.io/yf-jungbi/`
2. 크롬: 우측 메뉴(⋮) → **홈 화면에 추가**  /  아이폰 사파리: 공유 → **홈 화면에 추가**
3. 홈에 생긴 아이콘을 누르면 주소창 없는 앱처럼 실행됩니다(오프라인도 열림).

## 화면
- ✅ 딱 한 줄 결론(항상 보임) + ①점도 ②공임나라 준비물 3종 ③가격(쿠팡 검색) ④정비학습 ⑤순서
- 제목을 누르면 **접기/펼치기**, 우측 상단 **전체 펼치기/접기** 버튼.
- 가격은 클릭이 아니라 "쿠팡 > 「검색어」 검색 → 가격 ~이면 맞음" 안내.

## 구조 (나중에 내가 고칠 때)
- `index.html` `style.css` `app.js` — 화면/디자인/렌더 (다크 마스큘린)
- `data.json` — **내용은 여기만 고치면 됨**(app.js에 같은 내용 내장본도 있음)
- `manifest.json` `sw.js` `icon-*` — 앱 설치·오프라인용
- 이 웹앱은 데스크톱 도감과 **별도 리포(yf-jungbi)** 로 분리 관리됩니다.

조사일 2026-07-05 · 가격/공임은 시점·지점 따라 변동, 구매 전 품번·용량 재확인.
